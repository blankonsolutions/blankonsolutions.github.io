[특청 플랫폼 통합 설계도면]

1. 플랫폼 전체 구조 (시스템 전체 구조)
   - 진입 페이지: index.html (사용자 분류 및 입장)
   - 전용 페이지 A: customer.html (개인용 - 의뢰 및 업체 연결)
   - 전용 페이지 B: business.html (업체용 - 수주 및 현장 업무)
   - 전용 페이지 C: gov.html (공공용 - 행정 지원 및 모니터링)
   - 전용 페이지 D: admin.html (관리용 - 플랫폼 통합 관리)

--------------------------------------------------

2. 상세 메뉴 및 화면 구성 (메뉴 및 화면 구성)

[인덱스 진입 페이지 (index.html)]
├─ 메인 홍보 영역: 서비스 비전 및 브랜드 로고
├─ 포털 선택: 4대 전용 페이지 진입 버튼 (고객, 업체, 관공서, 관리자)
├─ 빠른 가입: 고객/업체 유형별 즉시 가입
├─ 소개: 기업 소개 및 소개 발표자료(PPT) 연결
└─ 현황 수치: 플랫폼 누적 데이터(작업건수, 인증비율)

[고객용 페이지 (customer.html)]
├─ 하단 메뉴바: [홈] [요청목록] [견적함] [내정보]
├─ 한눈에 보기(대시보드): 
│  ├─ 프리미엄 안심 케어 (본사 직접 관리 솔루션)
│  ├─ 무료 안심 견적 신청 (의뢰 양식)
│  ├─ 특수청소 가이드/팁 (전문 지식 창고)
│  └─ 심리 케어 연계 서비스 (마음 사후 관리)
├─ 견적 관리: 받은 견적 비교, 업체 정보 확인, 익명 상담 채팅
└─ 부가 기능: 알림 센터, 안심 보상 제도 안내, 고객 센터(자주 묻는 질문/1:1문의)

[업체용 페이지 (business.html)]
├─ 하단 메뉴바: [홈] [매출분석] [커뮤니티] [내등급]
├─ 한눈에 보기(대시보드):
│  ├─ 수익 관리: 이번 달 확정 수익 및 매출 성장 분석
│  ├─ 스마트 도구: 음성 현장 보고서(말하기로 작성), 폐기물 업체 조회
│  ├─ 비즈니스 연결: 장비 임대 및 인테리어 협력사 연결
│  └─ 운영 관리: 작업 실적 관리, 정산/서류 관리(문서 자동 생성)
├─ 현장 업무 도구: 위치 인증 사진 촬영, 음성 인식 AI 보고서 자동화
└─ 비즈니스 관리: 영업 서류 제출, 전문가 전용 게시판, 본사 제안창구

[관공서 행정 페이지 (gov.html)]
├─ 하단 메뉴바: [홈] [위험지도] [행정증빙] [문서창고]
├─ 한눈에 보기(대시보드):
│  ├─ 긴급 경보: 지역별 고독사 위험 지수 실시간 알림
│  ├─ 스마트 매칭: 행정 구역별 우수 업체 자동 추천
│  ├─ 소액 발주: 긴급 청소 및 수의계약 요청 시스템
│  └─ 진행 관리: 계약 단계별 현황 및 서류 확인 목록
├─ 현황 모니터링: 동네별 위험도 지도(히트맵) 및 AI 위험 분석
└─ 행정 부가 도구: 감사 자료 패키지 생성, 행정 데이터 저장소, 긴급 공지 전파

[마스터 관리자 페이지 (admin.html)]
├─ 하단 메뉴바: [홈] [업체심사] [운영설정] [전체기록]
├─ 한눈에 보기(대시보드): 플랫폼 가입 및 활동 지표, 주요 관리 도구
├─ 심사 시스템: 파트너 입점 심사 (서류 검토, 승인/반려 처리)
├─ 운영 관리: 이용약관 및 정책 관리, 플랫폼 설정
└─ 보안 및 기록: 시스템 전체 작업 기록(로그) 추적 및 분석

--------------------------------------------------

3. 핵심 데이터 흐름 (데이터 흐름)
   - 의뢰 흐름: [고객] 신청 -> [관리자/AI] 검토 -> [업체] 견적 발송
   - 현장 흐름: [업체] 음성/위치 보고 -> [관리자] 검증 -> [관공서] 행정 서류 수신
   - 정산 흐름: [고객] 결제 -> [관리자] 정산 처리 -> [업체] 수익금 입금
